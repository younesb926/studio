# **App Name**: Souk Electra

## Core Features:

- Product Catalog: Browse products by category (Petit électroménager, Gros électroménager, Informatique, Télévisions, Climatiseurs, Chauffe-eau, Téléphones & Tablettes).
- Product Search: Find products quickly using the search bar.
- Product Details: View product images, price, description, and availability.
- Shopping Cart: Add products to the cart and manage quantities.
- User Authentication: Register and login using Firebase Authentication.
- Cash on Delivery: Enable cash on delivery for customers in Morocco.
- AI-Powered Product Recommendations: AI Tool analyzes user browsing history and preferences to generate relevant product recommendations.

## Style Guidelines:

- Primary color: Vibrant yellow (#FFC107) to match the modern Moroccan style and add energy.
- Background color: Light grey (#F5F5F5) to create a clean and minimal look. 
- Accent color: Black (#212121) for text and UI elements, providing a strong contrast.
- Body and headline font: 'Inter', a sans-serif font for clear readability.
- Use a set of minimalist icons for categories and UI elements.
- Mobile-first responsive design with a clean grid layout for product listings.
- Smooth transitions and animations for UI elements like the slide-out menu and cart updates.
